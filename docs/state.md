## Где мы сейчас

Онбординг завершён. Это «Пульт руководителя» — отдельная страница (LEFT_MENU)
для директора и замов портала `alfa-prj.bitrix24.ru`: сводка по всем объектам
компании (иерархия смарт-процессов Сделка (1050) → Этап/Milestone (1054) →
Модуль (1062), плюс Оплаты (1058)) — сроки и ключевые показатели без
погружения в детали конкретной сделки. Бизнес-контекст полей, стадий и связей
между смартами — в `/source` (5 файлов: Deal, Milestone, Module, Pay, Задачи).
Код приложения (UI пульта, агрегация данных) ещё не написан — только каркас шаблона.
